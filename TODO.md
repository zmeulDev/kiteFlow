# KITEFLOW - SaaS ROADMAP

## Phase 1: Core Product (Completed)
- [x] Multi-tenancy isolation (TenantScope).
- [x] Kiosk check-in/out flow.
- [x] Host notifications (Email/DB).
- [x] Fast Pass / QR architecture.
- [x] Dashboard with Live Stats & Visitor Logs.
- [x] Brand/Settings customization.

## Phase 2: SaaS Transformation (Completed)
- [x] Custom Super Admin Panel at `/superadmin`.
- [x] Fully Responsive UI (Mobile-first sidebar & layouts).
- [x] Marketing Site (Landing Page) at `/`.
- [x] Self-service Signup & Tenant Registration at `/register`.

## Phase 3: Commercial & Enterprise (Ongoing)
- [x] Notification Center & History page.
- [x] Interactive Visitor Calendar.
- [x] Multi-Location Support (Manage wings/desks).
- [x] Meeting Room Management & Bookings.
- [x] Global Tenant Editing (Super Admin).
- [x] User Profile & Password Management.
- [x] Automated GDPR Data Pruning & Testing.
- [ ] Billing & Subscription integration (Stripe).
- [ ] Advanced Reporting (PDF downloads).

## Feature Backlog (Competitive Research)
- [ ] **Digital NDAs & E-Signatures**: Integrated signature pad on kiosk for legal compliance.
- [ ] **WhatsApp Host Notifications**: Instant arrival alerts via WhatsApp API.
- [ ] **Badge Printing**: Integration with thermal printers (Brother/Dymo).
- [ ] **Emergency Evacuation List**: One-tap real-time visitor manifest for safety drills.
- [ ] **Resource Booking**: Desk and parking spot reservation for employees/guests.
- [ ] **Delivery Management**: "Courier Mode" for package drop-offs and recipient alerts.
- [ ] **Employee/Contractor Flows**: Attendance tracking and safety inductions (videos/quizzes).
- [ ] **Occupancy Analytics**: Heatmaps and peak-time occupancy reporting.


## Technical Notes
- Stack: Laravel 12, Livewire 3, Alpine.js, SQLite.
- Design: SaaS Premium (Minimalist, Inter, Indigo/Slate).
