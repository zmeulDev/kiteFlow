Company Administrator Role
- to fully review the role and permission of company administrator
- 

Schedule visits
- start with: schedule visit date, time, number of people
- take in consideration if a building / entrance is active or not
- if a company has not active contract or expired contract, the visit should not be scheduled. check if there is an active contract for the company at first
- meeting room should be active and available of the visit date and time
- if number of people is selected, the meeting room should be able to accommodate the number of people
- 


Notifications
- implement notification system and take RBAC system into consideration
- have a dedicated notification page
- notification should be sent to the host when a visitor is scheduled
- notification should be sent to the visitor when a visit is scheduled
- notification should be sent to the host when a visitor is checked in
- notification should be sent to the visitor when a visit is checked out
- notification should be sent to the host when a visit is cancelled
- notification should be sent to the visitor when a visit is cancelled
- notification should be sent to the host when a visit is rescheduled
- notification should be sent to the visitor when a visit is rescheduled
- notification should be sent to the host when a visit is completed


Help Desk
- implement help desk system
- each location administrator can open tickets for company administrator
- help desk tickets will appear in notification page
